# 🔐 Sensitive Data Classifier

An AI-powered system for detecting and classifying sensitive information (PII/PHI) with anomaly detection capabilities.

## Features
- **Hybrid Detection**: Combines regex patterns with transformer embeddings
- **Anomaly Detection**: Identifies unusual data access patterns
- **REST API**: FastAPI-based endpoints with JWT authentication
- **Audit Logging**: Comprehensive logging with Elasticsearch integration
- **Compliance Ready**: Built for GDPR/DPDP Act compliance
